Kurulum ve Çalıştırma

Projeyi çalıştırmak için aşağıdaki 3 adımı uygulamanız yeterlidir:

1. Veritabanı Hazırlığı
* MySQL sunucunuzu başlatın.
* Proje içindeki `veritabani.sql` dosyasındaki kodları kopyalayıp SQL panelinde çalıştırın.
* Bu işlem `egitim_db` veritabanını ve `251109052_` ön ekli tabloları oluşturacaktır.

2. Bağımlılıkları Yükleme
Proje klasörünün içinde bir terminal açın ve şu komutu yazarak gerekli paketleri indirin:
bash
npm install
